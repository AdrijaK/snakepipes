RNA-seq¶

RNA-seq workflow v0.6 - MPI-IE workflow for RNA mapping and analysis

usage example:
RNA-seq -i input-dir -o output-dir mm10

usage: RNA-seq -i INDIR -o OUTDIR [-h] [-v] [-c CONFIGFILE]
               [--cluster_configfile CLUSTER_CONFIGFILE] [-j INT] [--local]
               [--snakemake_options STR] [--tempdir TEMPDIR] [--DAG] [-m MODE]
               [--downsample INT] [--trim_prg {cutadapt,trimgalore}] [--trim]
               [--trim_options STR] [--fastqc] [--library_type]
               [--mapping_prg STR] [--star_options STR] [--hisat_options STR]
               [--salmon_index_options STR] [--featurecounts_options STR]
               [--filter_annotation STR] [--DE SAMPLE_INFO] [--bw-binsize INT]
               [--dnaContam] [--plotFormat STR] [--VCFfile STR]
               [--strains STR] [--SNPfile STR] [--Nmasked_index STR]
               GENOME

Positional Arguments¶

GENOME genome acronym of target organism (supported: 'dm3', 'dm6', 'hs37d5', 'mm9', 'mm10', 'SchizoSPombe_ASM294v2')

Named Arguments¶

--plotFormat

Possible choices: png, pdf, None

Format of the output plots from deeptools. Select 'none' for no plot (default: 'None')

required arguments¶

-i, --input-dir
 input directory containing the FASTQ files, either paired-end OR single-end data
-o, --output-dir
 output directory

general arguments¶

-v, --verbose verbose output (default: 'None')
-c, --configfile
 configuration file: config.yaml. In absence of a config file, the default options from the workflows would be selected from /workflows/<workflow_name>/defaults.yaml (default: 'None')
--cluster_configfile
 configuration file for cluster usage. In absence, the default options from shared/cluster.yaml and workflows/[workflow]/cluster.yaml would be selected (default: 'None')
-j, --jobs maximum number of concurrently submitted Slurm jobs / cores if workflow is run locally (default: 'None')
--local

run workflow locally; default: jobs are submitted to Slurm queue (default: 'False')

Default: False

--snakemake_options
 Snakemake options to be passed directly to snakemake, e.g. use --snakemake_options='--dryrun --rerun-incomplete --unlock --forceall'. (default: 'None')
--tempdir used prefix path for temporary directory created via mktemp. Created temp dir gets exported as $TMPDIR and is removed at the end of this wrapper! (default: 'None')
--DAG

If specified, a file called pipeline.pdf is produced in the output directory that shows the rules used and their relationship to each other.

Default: False

Optional arguments¶

-m, --mode workflow running modes (available: 'mapping-free, mapping, allelic-mapping, deepTools_qc') (default: 'None')
--downsample downsample the given number of reads randomly from each FASTQ file (default: 'None')
--trim_prg

Possible choices: cutadapt, trimgalore

trimming program: cutadapt or trimgalore (default: 'None')

--trim Activate fastq read trimming. If activated, Illumina adaptors are trimmed by default. Additional parameters can be specified under --trim_options (default: 'None')
--trim_options Additional option string for trimming program of choice. (default: 'None')
--fastqc run FastQC read quality control (default: 'None')
--library_type user provided library type strand specificity. featurCounts style: 0, 1, 2 (Illumina TruSeq); default: 'None')
--mapping_prg Program used for mapping: STAR or HISAT2 (default: 'None')
--star_options STAR option string, e.g.: '--twopassMode Basic' (default: 'None')
--hisat_options
 HISAT2 option string (default: 'None')
--salmon_index_options
 Salmon index options, e.g. '--type fmd' (default: 'None')
--featurecounts_options
 featureCounts option string. The options '-p -B' are always used for paired-end data (default: 'None')
--filter_annotation
 filter annotation GTF by grep for use with Salmon, e.g. use --filter_annotation='-v pseudogene'; default: 'None')
--DE Information on samples (required for DE analysis); see 'snakemake_workflows/shared/tools/sampleInfo.example.tsv' for example. The column names in the tsv files are 'sample' and 'condition'.The first entry defines which group of samples are control. This way, the order of comparison and likewise the sign of values can be changed. The DE analysis might fail if your sample names begin with a number. So watch out for that! (default: 'None')
--bw-binsize bin size of output files in bigWig format (default: 'None')
--dnaContam Returns a plot which presents the proportion of the intergenic reads (default: 'None')

allele-specific mapping arguments¶

--VCFfile

VCF file to create N-masked genomes (default: 'None')

Default: ""

--strains

Name or ID of SNP strains separated by comma (default: 'None')

Default: ""

--SNPfile

File containing SNP locations (default: 'None')

Default: ""

--Nmasked_index
 

N-masked index of the reference genome (default: 'None')

Default: ""
